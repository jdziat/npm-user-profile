# NPM-user-PROFILE

Allows you to quickly and easily get your npm profile


## Install

```js

const NpmProfile = require('npm-user-profile')

```

## Usage

```js
(async () => {

const NpmProfile = require('npm-user-profile')
const profile = await NpmProfile.profile()
console.log(profile)

})
```
